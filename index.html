Our Specializations

Web Design for Everybody

This specialization offered by coursera you will all the basic concepts about web design, including HTML5, CSS3, JavaScript, Bootstrap, responsive web design and much more.

Introduction to HTML5

This course will introduce you to the basic concepts of using HTML5. You will learn how the internet works, and how to create the structure of a professional web page.

Introduction to CSS3

This course will introduce you to the basic concepts of using CSS3. You will learn apply styles and formats to your web site.

Interactivity with JavaScript

In this course you will take your website to the next level. This course will show you how to make your page user interactive.

Advanced Styling with Responsive Design

The final course of the specialization will teach you, how to make your website responsive. So that it can be displayed in different screen sizes without errors.
